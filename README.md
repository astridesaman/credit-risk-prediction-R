# credit-risk-prediction
Projet réalisé en R dans le cadre du cours d’Informatique Décisionnelle en L3 IA à l’Université Côte d’Azur. Il consiste à prédire le risque de défaut de paiement à partir de deux dataset. Plusieurs modèles de classification sont comparés afin de sélectionner le meilleur classifieur, ensuite utilisé pour prédire les défauts sur de nouveaux clients.
